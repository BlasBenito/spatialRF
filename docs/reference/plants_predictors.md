# Predictor variable names for plant richness examples

Character vector of predictor variable names from
[plants_df](https://blasbenito.github.io/spatialRF/reference/plants_df.md)
(columns 5 to 21).

## Usage

``` r
data(plants_predictors)
```

## Format

A character vector of length 17.
