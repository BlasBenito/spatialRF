# Matrix of distances among ecoregion edges.

Distance matrix (in km) among the edges of the American ecoregions
described in the
[plant_richness_df](https://blasbenito.github.io/spatialRF/reference/plant_richness_df.md)
dataset.

## Usage

``` r
data(distance_matrix)
```

## Format

A numeric matrix with 227 rows and columns.
