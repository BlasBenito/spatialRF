# Coordinates for plant richness data

Spatial coordinates (longitude and latitude) extracted from
[plants_df](https://blasbenito.github.io/spatialRF/reference/plants_df.md)
for use in spatial modeling functions.

## Usage

``` r
data(plants_xy)
```

## Format

A data frame with 227 rows and 2 columns:

- `x`: Longitude in degrees (WGS84).

- `y`: Latitude in degrees (WGS84).
