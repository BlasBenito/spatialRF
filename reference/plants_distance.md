# Distance matrix between ecoregion edges

Distance matrix (in km) between the edges of American ecoregions in
[plants_df](https://blasbenito.github.io/spatialRF/reference/plants_df.md).

## Usage

``` r
data(plants_distance)
```

## Format

Numeric matrix with 227 rows and 227 columns.
