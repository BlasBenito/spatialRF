# Response variable name for plant richness examples

Character string containing the name of the response variable in
[plants_df](https://blasbenito.github.io/spatialRF/reference/plants_df.md):
"richness_species_vascular".

## Usage

``` r
data(plants_response)
```

## Format

A character string of length 1.
